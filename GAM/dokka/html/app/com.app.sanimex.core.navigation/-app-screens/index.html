AppScreens

sealed class AppScreens

Clase sellada que define las rutas de navegación de la aplicación.

Cada objeto dentro de esta clase representa una pantalla diferente de la aplicación y contiene la ruta de navegación asociada. Esto facilita la gestión y el uso de las rutas dentro del sistema de navegación de Compose.
