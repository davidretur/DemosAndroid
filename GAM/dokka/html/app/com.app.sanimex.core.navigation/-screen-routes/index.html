ScreenRoutes

object ScreenRoutes

Objeto que contiene las constantes de las rutas de navegación de la aplicación.

Este objeto centraliza la definición de todas las rutas de la aplicación como constantes de cadena. Esto ayuda a garantizar la coherencia y facilita la refactorización de las rutas en caso de ser necesario.
