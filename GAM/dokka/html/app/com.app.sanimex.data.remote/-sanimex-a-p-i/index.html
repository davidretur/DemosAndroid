interface SanimexAPI

Interfaz que define los endpoints de la API de Sanimex utilizando Retrofit.

Cada función suspendida en esta interfaz representa una llamada a un endpoint específico de la API, indicando el método HTTP utilizado (POST, GET, DELETE) y la ruta relativa. Los parámetros de la función se corresponden con los parámetros de la consulta o el cuerpo de la petición.
