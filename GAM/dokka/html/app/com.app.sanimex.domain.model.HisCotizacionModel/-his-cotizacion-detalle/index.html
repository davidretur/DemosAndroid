HisCotizacionDetalle

data class HisCotizacionDetalle(val codebar: String, val description: String, val cantidad: String, val precioUnitario: Float, val status: String, val nombreCliente: String, val clasificacion: String)

Data class que representa el detalle de un item en el histórico de cotizaciones.

Contiene información detallada sobre un producto incluido en una cotización histórica, como su código de barras, descripción, cantidad, precio unitario, estado, el nombre del cliente asociado y su clasificación.
