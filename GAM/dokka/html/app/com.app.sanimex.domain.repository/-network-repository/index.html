NetworkRepository

interface NetworkRepository

Interfaz que define las operaciones para interactuar con la capa de red de la aplicación Sanimex.

Cada función suspendida en esta interfaz representa una llamada a un endpoint específico de la API, devolviendo un Flow de Resource que encapsula el estado de la petición (cargando, éxito o fallo) y los datos o el mensaje de error correspondientes.
