MainButton

@Preview
@Composable
fun MainButton(modifier: Modifier = Modifier.height(58.dp), text: String? = null, icon: Int? = null, cornerRadius: Dp = 5.dp, padding: Dp = 16.dp, onClick: () -> Unit = {}, isLoading: Boolean = false, isEnabled: Boolean = true, activeColor: Color = MaterialTheme.colorScheme.primary, inactiveColor: Color = MaterialTheme.colorScheme.tertiary)

Composable que representa un botón principal personalizado con soporte para texto, ícono y estado de carga.

Este botón utiliza los estilos del tema de Material Design y permite configurar su apariencia (color, forma, habilitado/deshabilitado) y comportamiento (acción al hacer clic, indicador de carga).

Author

David Duarte

Parameters

modifier

Modificador para personalizar el layout del botón (por defecto, establece una altura de 58 dp).

text

Texto a mostrar en el botón (opcional).

icon

Recurso de ícono (Int) a mostrar en el botón (opcional).

cornerRadius

Radio de las esquinas redondeadas del botón (por defecto, 5 dp).

padding

Espacio interno alrededor del contenido del botón (por defecto, 16 dp).

onClick

Función lambda que se ejecuta al hacer clic en el botón (por defecto, una función vacía).

isLoading

Booleano que indica si el botón se encuentra en estado de carga (por defecto, false). Si es true, se muestra una animación de carga de Lottie.

isEnabled

Booleano que indica si el botón está habilitado para recibir clics (por defecto, true).

activeColor

Color de fondo del botón cuando está habilitado (por defecto, el color primario del tema).

inactiveColor

Color de fondo del botón cuando está deshabilitado (por defecto, el color terciario del tema).
